# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus on the technical work, and
keep the discussion welcoming to newcomers and experienced contributors alike.

## Reporting

If you observe unacceptable behavior, contact the maintainers privately by
opening a confidential issue or emailing the address in the GitHub profile.
Reports will be handled discreetly and in good faith.
